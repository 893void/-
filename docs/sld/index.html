単線結線図

単線結線図（Single Line Diagram）から、各機器の役割と接続を学びます。

単線結線図とは

単線結線図は、電気設備の構成を1本の線で簡略化して表した図面です。

  • 三相3線の電路も1本の線で表す
  • 機器は記号（シンボル）で表す
  • 電気の流れと保護協調を把握するための基本図面

図から学ぶ

下の図の機器をクリックすると、各機器の解説ページに移動します。
