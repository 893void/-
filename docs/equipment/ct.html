CT（変流器）

CT（変流器）

概要

CT（Current Transformer） は、大きな電流を計測可能な小さな電流に変換する変成器。

高圧回路の電流を直接測定するのは危険なため、CTで変換して計器や継電器に入力する。

二次側の定格電流は 5A が標準。

基礎知識

動作原理

CTは変圧器と同じ電磁誘導の原理を利用している。

一次側（高圧回路）に電流が流れる

鉄心に磁束が発生

二次側に電流が誘導される

変流比

I₁ / I₂ = N₂ / N₁
  • I₁：一次電流
  • I₂：二次電流
  • N₁：一次巻数
  • N₂：二次巻数

例：変流比 100/5A のCT → 一次側100Aで二次側5A

構造

部位 説明
鉄心 磁束を通す
一次巻線 高圧回路に接続（または貫通型）
二次巻線 計器・継電器に接続
二次端子 K, L端子（極性あり）
銘板 変流比、定格負担等を記載

種類

構造による分類

種類 特徴
巻線形CT 一次・二次とも巻線がある
貫通形CT 一次導体を鉄心に貫通させる
ブッシング形CT 機器のブッシングに組み込み

用途による分類

用途 特徴
計器用CT 精度重視、電流計・電力計に使用
保護用CT 過電流耐量重視、継電器に使用

二次側開放厳禁

なぜ危険か

CTの二次側を開放すると、以下の現象が起こる。

二次電流がゼロになる

一次電流による磁束を打ち消す電流がなくなる

鉄心が飽和し、異常な高電圧が発生

絶縁破壊、感電、火災の危険

対策

  • 二次側には必ず負荷（計器、継電器）を接続
  • 計器を外すときは 先に二次側を短絡 してから外す
  • 使用しない二次巻線は短絡しておく

鉄則：CTの二次側は絶対に開放しない

極性

極性とは

CTには極性（方向性）がある。

  • K端子（一次側）：電源側
  • L端子（一次側）：負荷側
  • k端子（二次側）：K端子と同極性
  • l端子（二次側）：L端子と同極性

極性の重要性

継電器や電力計に接続するとき、極性を間違えると：

  • 電力計が逆振れする
  • 継電器が誤動作する
  • 保護協調が崩れる
