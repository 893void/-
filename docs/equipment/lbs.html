LBS（高圧負荷開閉器）

概要

LBS（Load Break Switch） は、負荷電流を開閉できる高圧開閉器。

変圧器やコンデンサの一次側に設置され、PF（高圧限流ヒューズ） と組み合わせて保護する。

短絡電流は遮断できない。

基礎知識

役割

負荷の開閉：変圧器やコンデンサの入切

回路の区分：点検時の電路切り離し

PFと組み合わせて保護：短絡保護はPFが担当

構造

部位 説明
開閉部 負荷電流を開閉する接点
消弧室 アークを消弧する機構
ヒューズホルダ PFを取り付ける部分
操作ハンドル 手動開閉用
ストライカ PF溶断時にLBSを開放する機構

なぜLBS + PFの組み合わせか

電流の種類 担当
負荷電流 LBSで開閉
短絡電流 PFで遮断

VCBより安価で、小規模な分岐回路に適している。

種類

構造による分類

種類 特徴
限流ヒューズ付LBS PF内蔵、一般的
ヒューズなしLBS 別途保護装置が必要

用途による分類

用途 説明
変圧器用 TRの一次側保護
コンデンサ用 SCの開閉・保護
フィーダ用 配電回路の区分

PF（高圧限流ヒューズ）

PFとは

PF（Power Fuse） は、短絡電流を限流して遮断するヒューズ。

LBSと組み合わせて使用する。

特徴

項目 内容
限流効果 短絡電流のピーク値を抑制
遮断速度 極めて高速（数ミリ秒）
再使用 不可（溶断後は交換）

ストライカ方式

PFが溶断すると、内蔵のストライカ（突き出しピン）が飛び出し、LBSを自動的に開放する。

目的

  • 三相のうち1相だけPFが溶断すると欠相状態になる
  • 欠相は機器を損傷させるため、全相を開放する必要がある

VCBとの使い分け

項目 LBS + PF VCB
遮断能力 負荷電流のみ（短絡はPF） 短絡電流も遮断可能
価格 安い 高い
保護方式 PFで短絡保護 継電器と組み合わせ
再閉路 PF交換が必要 再投入可能
用途 分岐回路、小規模設備 主遮断装置

選定の目安

  • 主遮断装置 → VCB
  • 変圧器・コンデンサの一次側 → LBS + PF
  • コスト重視の小規模設備 → LBS + PF

操作上の注意

開閉順序

投入時

LBSを投入

二次側の開閉器を投入

開放時

二次側の開閉器を開放

LBSを開放

重要：LBSは負荷電流しか遮断できないため、大電流を遮断しようとすると接点が損傷する
