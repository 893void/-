変圧器

変圧器（TR）

概要

変圧器（Transformer） は、電磁誘導を利用して交流電圧を変換する機器。

高圧受電設備では、電力会社から受電した 6,600V を 200V/100V に変換して需要家に供給する。

略称は TR。

基礎知識

動作原理

変圧器は 電磁誘導の法則（ファラデーの法則）を利用している。

一次コイルに交流電流を流す

鉄心に交番磁束が発生

二次コイルに誘導起電力が生じる

電圧比 = 巻数比 の関係がある。

V₁ / V₂ = N₁ / N₂
  • V₁：一次電圧
  • V₂：二次電圧
  • N₁：一次巻数
  • N₂：二次巻数

構造（油入変圧器）

油入変圧器は 本体（タンク）と蓋が分離した構造 になっている。

部位 説明
タンク 鉄心とコイルを収納
絶縁油 絶縁と冷却を担う
ブッシング 一次・二次端子の引き出し部
フィン（放熱板） 外周に設置、放熱用
銘板 定格容量、電圧、製造年等を記載

種類

油入変圧器とモールド変圧器

絶縁方式 絶縁油 エポキシ樹脂
冷却方式 油の対流（自冷式が多い） 空気の自然対流
特徴 安価、実績多い 難燃性、保守容易

結線方式

結線 用途
Δ-Δ（デルタ-デルタ） 動力専用

サーモマーカとサーモラベル

サーモマーカ

  • 形状：塗料タイプ（ペン状）
  • 温度：単一温度（80℃など）
  • 貼付箇所：端子ボルト、接続部
  • 目的：接触不良による局所発熱の検知

サーモラベル

  • 形状：シールタイプ
  • 温度：複数温度（75/80/85℃など）
  • 貼付箇所：本体中央部
  • 目的：機器全体の温度上昇を監視

なぜ本体の「真ん中」に貼るか

油入変圧器の内部では、絶縁油が対流で循環している。

  • 上部：加熱された油が上昇 → 常に温度が高い
  • 下部：冷却された油が下降 → 温度が低い
  • 中央部：油の平均的な温度

最上部に貼ると常に高温を示し、過負荷の判断ができない。中央部なら「いつもより高い＝異常」と判断できる。

吹き出し痕

吹き出し痕とは

変圧器の絶縁油が外部に漏れ出た跡。

吹き出し痕が示すこと

  • 過去に 過負荷運転 があった可能性
  • 油温が異常上昇し、内圧が上がった証拠
  • 絶縁油が減少・劣化している可能性

確認すべき箇所

  • 上蓋と本体タンクのつなぎ目（最も漏れやすい）
  • ブッシングの根元
  • ドレンバルブ周辺
  • フィンの汚れ具合も参考になる

発見した場合の対応

記録：発見箇所、範囲、写真撮影

報告：点検報告書に記載、お客様への報告

提案：経過観察 / 油中ガス分析 / 絶縁油交換 / 更新

※具体的な対応手順は社内規程に従う

異音の種類と原因

変圧器の正常な運転音は「ブーン」という低いうなり音（磁歪音）。

音の種類 考えられる原因
うなり音が大きくなった 過負荷、鉄心の緩み
ジジジ、パチパチ音 放電（絶縁劣化の可能性）
カタカタ音 カバー・部品の緩み
ボコボコ音 油中の気泡（異常発熱の可能性）

透明カバーがずれると異音が発生する理由

変圧器の振動（磁歪振動）がカバーに伝わり、共振してカタカタと音が鳴る。

接地

変圧器には B種接地 が施される。

B種接地の目的

  • 混触時（高圧が低圧に侵入）の低圧側電位上昇を抑制
  • 地絡電流の帰路を確保

ZCTによる監視

  • B種接地線をZCT（零相変流器）に貫通させる
  • 低圧側で漏電が発生 → B種接地線に電流が流れる
  • ZCTがこの電流を検出 → 監視装置（Esnavi等）に信号
  • 常時監視により漏電の早期発見が可能

点検時の確認

接地線がZCTを 正しく貫通しているか を確認する。工事後などにZCTを通っていないと漏電を検出できない。
