VT（計器用変圧器）

概要

VT（Voltage Transformer） は、高電圧を計測可能な低電圧に変換する変成器。

PT（Potential Transformer） とも呼ばれる。

高圧回路の電圧を直接測定するのは危険なため、VTで変換して計器や継電器に入力する。

二次側の定格電圧は 110V が標準。

基礎知識

動作原理

VTは変圧器と同じ電磁誘導の原理を利用している。

一次側に高電圧を印加

鉄心に磁束が発生

二次側に低電圧が誘導される

変圧比

V₁ / V₂ = N₁ / N₂

例：変圧比 6,600/110V のVT → 一次側6,600Vで二次側110V

構造

部位 説明
鉄心 磁束を通す
一次巻線 高圧回路に接続
二次巻線 計器・継電器に接続
二次端子 極性あり
銘板 変圧比、定格負担等を記載

種類 特徴

二次側短絡厳禁

なぜ危険か

VTの二次側を短絡すると、以下の現象が起こる。

二次側に過大な電流が流れる

VTが焼損する

一次側ヒューズが溶断（設備によっては停電）

対策

  • 二次側は絶対に短絡しない
  • 二次側の配線作業時は必ず一次側を開放
  • 使用しない二次巻線は開放しておく

CTとの違い

機器 禁止事項
CT 二次側 開放 厳禁
VT 二次側 短絡 厳禁

VTヒューズ

役割

VTの一次側または二次側にはヒューズが設置されている。

  • 一次側ヒューズ：VT内部短絡時の保護
  • 二次側ヒューズ：二次回路短絡時の保護

ヒューズ溶断時の影響

  • 電圧計がゼロを指示
  • 電力計、電力量計が動作しなくなる
  • 電圧に関連する継電器（UVR等）が誤動作の可能性

関連機器との組み合わせ

組み合わせ 用途

VT + CT + 電力量計 電力量計量

VT + OVR 過電圧保護
