保護継電器の基礎

概要

保護継電器とは、電力系統に発生する異常（過電流、地絡、電圧異常など）を検出し、遮断器に遮断信号を送ることで系統を保護する装置です。

電力系統の安全を守る「番人」として、24時間365日稼働し続けています。

図解

異常検出 → 継電器動作 → 遮断器トリップ → 系統保護

詳細説明

なぜ保護継電器が必要なのか

電力系統では、落雷、機器故障、絶縁劣化などにより、様々な異常が発生する可能性があります。これらの異常を放置すると、機器の損傷、火災、広範囲の停電につながります。

保護継電器は、これらの異常を瞬時に検出し、故障箇所を系統から切り離すことで、被害の拡大を防ぎます。

保護継電器の基本動作

保護継電器は、以下の流れで動作します。

検出：CT（変流器）やVT（計器用変圧器）から電流・電圧情報を受け取る

判定：設定値と比較し、異常かどうかを判断する

動作：異常と判定した場合、遮断器にトリップ信号を送る

遮断：遮断器が開放され、故障箇所が系統から切り離される

保護継電器の種類

保護継電器は、検出する異常の種類によって分類されます。

  • 過電流継電器（OCR）：電流が設定値を超えた場合に動作
  • 地絡継電器（GR）：地絡（漏電）を検出して動作
  • 方向性地絡継電器（DGR）：地絡電流の方向を判別して動作
  • 地絡過電圧継電器（OVGR）：零相電圧が設定値を超えた場合に動作
  • 不足電圧継電器（UVR）：電圧が設定値を下回った場合に動作
