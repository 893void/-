# UVR（不足電圧継電器）

## 概要

不足電圧継電器（UVR：Under Voltage Relay）は、電圧が設定値を下回った場合に動作する保護継電器です。

電圧低下を検出し、機器の保護や系統の安定化のための動作を行います。

## 図解

```
VT → 電圧検出 → 設定値比較 → 動作 → 保護動作
```

## 詳細説明

### 動作原理

UVRは、VT（計器用変圧器）から電圧情報を受け取り、予め設定された電圧値と比較します。電圧が設定値を下回ると、設定された時間経過後に動作します。

### UVRの用途

UVRは、主に以下の用途で使用されます。

- **電動機保護**：電圧低下時の過電流を防止するため、電動機を停止させる
- **自家発電設備**：商用電源の停電検出と自家発電への切替
- **系統連系保護**：分散電源の単独運転防止

### 整定項目

UVRの主な整定項目は以下の通りです。

- **動作電圧**：動作する電圧値の設定（通常80〜90%程度）
- **動作時間**：検出から動作までの時間設定
- **復帰電圧**：動作状態から復帰する電圧値の設定
