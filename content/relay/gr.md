# GR（地絡継電器）

## 概要

地絡継電器（GR：Ground Relay）は、地絡（漏電）を検出して動作する保護継電器です。

零相変流器（ZCT）を用いて零相電流を検出し、地絡事故から系統と人身を保護します。

## 図解

```
ZCT → 零相電流検出 → 設定値比較 → 動作 → 遮断器トリップ
```

## 詳細説明

### 動作原理

正常時、三相の電流はベクトル的に打ち消し合い、ZCTの出力は零になります。地絡が発生すると、この平衡が崩れて零相電流が流れ、GRが検出・動作します。

### GRとDGRの違い

GRは零相電流のみで動作を判断しますが、DGR（方向性地絡継電器）は零相電流と零相電圧の位相関係から地絡方向を判別します。GRは自家用構内の地絡保護に使用されることが多いです。

### 整定項目

GRの主な整定項目は以下の通りです。

- **感度電流**：動作する零相電流値の設定（通常0.2A程度）
- **動作時間**：検出から動作までの時間設定
