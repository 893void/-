# OVGR（地絡過電圧継電器）

## 概要

地絡過電圧継電器（OVGR：Over Voltage Ground Relay）は、零相電圧が設定値を超えた場合に動作する保護継電器です。

系統全体の地絡発生を検出し、警報や保護動作のトリガーとして使用されます。

## 図解

```
ZPD → 零相電圧検出 → 設定値比較 → 動作 → 警報/遮断
```

## 詳細説明

### 動作原理

電力系統で地絡が発生すると、零相電圧（V0）が発生します。OVGRは、ZPD（零相電圧検出器）からの零相電圧を監視し、設定値を超えると動作します。

### OVGRの特徴

OVGRは零相電圧のみで動作を判断するため、地絡の発生は検出できますが、地絡の方向（自系統か他系統か）は判別できません。そのため、警報用や、DGRと組み合わせた保護協調に使用されることが多いです。

### 整定項目

OVGRの主な整定項目は以下の通りです。

- **零相電圧整定値**：動作する零相電圧値の設定（通常5〜10%程度）
- **動作時間**：検出から動作までの時間設定
