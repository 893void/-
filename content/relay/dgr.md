# DGR（方向性地絡継電器）

## 概要

方向性地絡継電器（DGR：Directional Ground Relay）は、地絡電流の方向を判別して動作する保護継電器です。

零相電流と零相電圧の位相関係から地絡点の方向を特定し、自系統の地絡のみを選択的に遮断します。

## 図解

```
ZCT + ZPD → 位相比較 → 方向判定 → 動作 → 遮断器トリップ
```

## 詳細説明

### 動作原理

DGRは、ZCT（零相変流器）で零相電流を、ZPD（零相電圧検出器）で零相電圧を検出します。この2つの位相関係を比較することで、地絡が自系統側で発生したか、他系統側で発生したかを判別します。

### なぜ方向性が必要なのか

高圧配電系統では、他の需要家や配電線での地絡事故でも、零相電流が流れる場合があります。DGRを使用することで、自系統の地絡のみを検出し、不要な遮断（もらい事故）を防止できます。

### 整定項目

DGRの主な整定項目は以下の通りです。

- **零相電流整定値**：動作する零相電流値の設定
- **零相電圧整定値**：動作する零相電圧値の設定
- **動作時間**：検出から動作までの時間設定
