# ZPD（零相電圧検出器）

## 概要

**ZPD（Zero-phase Potential Device）** は、零相電圧（V₀）を検出する機器。

DGR（方向性地絡継電器）やOVGR（地絡過電圧継電器）と組み合わせて使用する。

---

## 基礎知識

### 零相電圧とは

正常な三相回路では、三相の電圧のベクトル和はゼロになる。

```
Va + Vb + Vc = 0（正常時）
```

地絡が発生すると、地絡相の対地電圧が低下し、健全相の対地電圧が上昇する。この不平衡により零相電圧が発生する。

```
Va + Vb + Vc = 3V₀（地絡時）
```

### なぜ零相電圧が必要か

**地絡方向の判別**

ZCTの零相電流だけでは、地絡が「自分の構内」なのか「配電線側」なのか判別できない。

ZCTの零相電流（I₀）とZPDの零相電圧（V₀）の位相関係を見ることで、地絡の方向を判別できる。

---

## 動作原理

### コンデンサ分圧方式

ZPDは三相のコンデンサを利用して零相電圧を検出する。

1. 各相にコンデンサを接続（対地間）
2. 三相のコンデンサの中性点に零相電圧が現れる
3. この電圧を継電器に入力

### 検出回路

```
R相 ──[C]──┐
S相 ──[C]──┼── V₀（零相電圧）
T相 ──[C]──┘
```

正常時：三相平衡のため V₀ ≒ 0  
地絡時：不平衡により V₀ が発生

## 種類

### 構造による分類

| 種類 | 特徴 |
|------|------|
| ZPD | コンデンサ分圧式、小型 |
| GPT（EVT） | 接地形計器用変圧器、高精度 |

### ZPDとGPTの違い

| 項目 | ZPD | GPT（EVT） |
|------|-----|-----------|
| 構造 | コンデンサ分圧 | 変圧器式 |
| 精度 | やや低い | 高い |
| サイズ | 小型 | 大型 |
| コスト | 安い | 高い |
| 用途 | 一般的な需要家 | 高精度が必要な場合 |

## 継電器との組み合わせ

| 組み合わせ | 保護内容 |
|-----------|---------|
| ZCT + ZPD + DGR | 方向性地絡保護 |
| ZPD + OVGR | 地絡過電圧保護 |

### DGR（方向性地絡継電器）との接続

- ZCT：零相電流（I₀）を検出
- ZPD：零相電圧（V₀）を検出
- DGR：I₀とV₀の位相関係から地絡方向を判別

### OVGR（地絡過電圧継電器）との接続

- ZPD：零相電圧（V₀）を検出
- OVGR：V₀が設定値を超えると動作

OVGRは主にPAS（SOG付）と組み合わせて使用する。

---

## 設置上の注意

### 接続相の確認

ZPDは三相全てに接続する必要がある。一相でも接続が外れると正常に動作しない。

### 接地の確認

ZPDの接地端子は確実に接地すること。接地が不完全だと誤動作の原因になる。

## トラブル事例

### 誤動作の原因

| 原因 | 対策 |
|------|------|
| 接地不良 | 接地状態を確認 |
| 配線誤り | 配線を確認 |
| ZPD故障 | 試験で確認 |

### 不動作の原因

| 原因 | 対策 |
|------|------|
| 接続外れ | 接続状態を確認 |
| ZPD故障 | 試験で確認 |
| 継電器故障 | 継電器試験で確認 |
