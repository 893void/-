# VCB（真空遮断器）

## 概要

**VCB（Vacuum Circuit Breaker）** は、真空中でアークを消弧して電路を遮断する開閉器。

高圧受電設備の **主遮断装置** として使用され、短絡電流や過電流を遮断して設備を保護する。

---

## 基礎知識

### 動作原理

1. 遮断指令を受ける（継電器動作、手動操作など）
2. 真空バルブ内の接点が開離
3. アーク（電弧）が発生
4. 真空中ではアークを維持する媒体がないため、急速に消弧
5. 電流が遮断される

**真空の特性**

- 絶縁耐力が高い
- アーク消弧能力が優れている
- 接点の消耗が少ない

### 構造

| 部位 | 説明 |
|------|------|
| 真空バルブ | 真空中に接点を封入した容器 |
| 操作機構 | 電磁操作式、ばね操作式など |
| トリップコイル | 遮断指令を受けて動作 |
| 投入コイル | 投入指令を受けて動作 |
| 開閉表示器 | 入/切の状態を表示 |
| 操作回数カウンタ | 累積開閉回数を表示 |

---

## 種類

### 操作方式による分類

| 方式 | 特徴 |
|------|------|
| 電磁操作式 | 電磁石で直接操作、構造がシンプル |
| ばね蓄勢式 | ばねにエネルギーを蓄えて操作、大容量向け |

### 設置場所による分類

| 種類 | 用途 |
|------|------|
| 受電用VCB | 主遮断装置として使用 |
| フィーダ用VCB | 配電盤内で分岐回路の保護 |

## 月次点検のポイント

### チェックリスト

| No. | 確認項目 | 確認内容 |
|-----|---------|---------|
| 1 | 開閉表示 | 表示が実際の状態と一致しているか |
| 2 | 操作回数カウンタ | 前回からの増加を記録 |
| 3 | 外観 | 損傷、変形、腐食がないか |
| 4 | 異音 | 異常な音がしていないか |
| 5 | 異臭 | 焦げ臭がしていないか |
| 6 | 表示灯 | 正常に点灯しているか |
| 7 | 操作電源 | 制御電源が正常か |

### 操作回数カウンタの確認

VCBには寿命がある。操作回数を記録し、メーカー推奨の点検・交換時期を把握する。

| 目安 | 内容 |
|------|------|
| 定期点検推奨 | 1,000〜2,000回程度 |
| 真空バルブ交換目安 | 10,000〜20,000回程度 |

※機種により異なるため、メーカー仕様を確認すること

## 年次点検のポイント

### 月次点検との違い

| 項目 | 月次点検 | 年次点検 |
|------|---------|---------|
| 停電 | 不要 | 必要 |
| 絶縁抵抗測定 | しない | する |
| 動作試験 | しない | する |
| 清掃 | しない | する |
| 接点確認 | しない | する（可能な機種） |

### 年次点検での確認項目

| 項目 | 内容 |
|------|------|
| 絶縁抵抗測定 | 主回路、制御回路 |
| 動作試験 | 投入・遮断動作の確認 |
| インターロック確認 | DS連動など |
| 清掃 | 碍子、外箱の清掃 |
| 増し締め | 端子部の締め付け確認 |

## 継電器との連携

VCBは単独では保護機能を持たない。継電器からの指令で動作する。

### 代表的な組み合わせ

| 継電器 | 保護内容 |
|--------|---------|
| OCR | 過電流保護 |
| GR | 地絡保護（高圧側） |
| DGR | 方向性地絡保護 |
| UVR | 不足電圧保護 |

### 保護連動試験

年次点検では、継電器が動作したときにVCBが確実に遮断するか確認する。

---

## OCR・GRとの一体型

小規模設備では、VCB本体にOCRやGRが内蔵された一体型が使用されることがある。

| 種類 | 特徴 |
|------|------|
| VCB単体 | 継電器は別置き |
| OCR付VCB | 過電流継電器内蔵 |
| GR付VCB | 地絡継電器内蔵 |

---

## LBSとの違い

| 項目 | VCB | LBS |
|------|-----|-----|
| 遮断能力 | 短絡電流を遮断可能 | 負荷電流のみ遮断可能 |
| 用途 | 主遮断装置 | 負荷開閉、変圧器一次側 |
| 価格 | 高い | 安い |
| 保護機能 | 継電器と組み合わせ | PF（ヒューズ）と組み合わせ |
