# PAS（高圧気中開閉器）

## 概要

**PAS（Pole mounted Air Switch）** は、電柱や引込柱に設置される高圧気中開閉器。

需要家の **責任分界点** に設置され、波及事故防止の重要な役割を担う。

SOG（Storage Over current Ground）機能付きが一般的。

---

## 基礎知識

### 役割

1. **責任分界点の明確化**：電力会社と需要家の境界
2. **波及事故防止**：構内事故を自動遮断し、配電線への波及を防ぐ
3. **区分開閉**：点検時の電路切り離し

### 構造

| 部位 | 説明 |
|------|------|
| 開閉部 | 気中で開閉する接点 |
| 碍子 | 絶縁支持物 |
| 操作ハンドル | 手動開閉用（フック棒で操作） |
| SOG制御装置 | 過電流・地絡検出と遮断制御 |
| 方向性SOG | DGR機能内蔵型 |

### SOG機能とは

**SOG = Storage Over current Ground**

- **S（Storage）**：蓄勢。ばねにエネルギーを蓄える
- **O（Over current）**：過電流検出
- **G（Ground）**：地絡検出

構内で短絡や地絡が発生すると、配電線の再閉路前に自動でPASを開放する。

---

## 種類

### 機能による分類

| 種類 | 機能 |
|------|------|
| 普通PAS | 手動開閉のみ |
| SOG付PAS | 過電流・地絡で自動遮断 |
| 方向性SOG付PAS | DGR機能内蔵、もらい事故防止 |
| UGS | 地中線用（キャビネット収納型） |

### 方向性SOGの重要性

**もらい事故とは**

他の需要家で地絡事故が発生したとき、自分の設備のPASが誤動作して停電すること。

**方向性SOGで解決**

- 電流の方向を判別
- 「自分の構内方向」の事故のみ遮断
- 「配電線方向」の事故では動作しない

## 月次点検のポイント

### チェックリスト

| No. | 確認項目 | 確認内容 |
|-----|---------|---------|
| 1 | 外観 | 損傷、変形、腐食がないか |
| 2 | 碍子 | 汚損、ひび割れがないか |
| 3 | 開閉表示 | 表示が実際の状態と一致しているか |
| 4 | 引込線 | 断線、たるみがないか |
| 5 | 鳥害 | 鳥の巣、糞害がないか |
| 6 | 樹木接触 | 樹木が接近していないか |
| 7 | SOG表示灯 | 正常に点灯しているか（地上設置の場合） |

### 碍子の汚損

碍子が汚損すると絶縁が低下し、地絡事故の原因になる。

| 環境 | 汚損の原因 |
|------|-----------|
| 海岸部 | 塩分付着 |
| 工場周辺 | 粉塵、煤煙 |
| 交通量の多い場所 | 排気ガス |

汚損がひどい場合は清掃または碍子交換を検討。

### 月次点検との違い

### 年次点検での確認項目

| 項目 | 内容 |
|------|------|
| 絶縁抵抗測定 | 主回路の絶縁抵抗 |
| SOG動作試験 | 過電流・地絡での動作確認 |
| 手動開閉試験 | フック棒での開閉確認 |

### SOG動作試験

SOG制御装置が正常に動作するか確認する。

1. 試験器を接続
2. 過電流試験：設定値で動作するか
3. 地絡試験：設定値で動作するか
4. 動作時間の確認

## 波及事故防止

### 波及事故とは

需要家の構内事故が原因で、配電線全体が停電すること。

**影響**

- 同じ配電線の他の需要家が停電
- 電力会社からの損害賠償請求の可能性
- 社会的信用の低下

### PASの役割

1. 構内で事故発生
2. 配電線の保護装置が動作（一旦停電）
3. 配電線が再閉路する **前に** PASが開放
4. 構内だけが切り離され、配電線は復旧

**重要**：PASが正常に動作しないと波及事故になる

---

## UGS（地中線用）

地中引込の場合は **UGS（Underground Gas Switch）** を使用。

| 項目 | PAS | UGS |
|------|-----|-----|
| 設置場所 | 電柱（架空） | 地上キャビネット |
| 絶縁方式 | 気中 | SF6ガス |
| 用途 | 架空引込 | 地中引込 |
