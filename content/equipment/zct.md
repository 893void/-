# ZCT（零相変流器）

## 概要

**ZCT（Zero-phase Current Transformer）** は、三相一括で零相電流（地絡電流）を検出する変流器。

地絡継電器（GR、DGR）と組み合わせて **地絡保護** に使用する。

---

## 基礎知識

### 零相電流とは

正常な三相回路では、三相の電流のベクトル和はゼロになる。

```
Ia + Ib + Ic = 0（正常時）
```

地絡（漏電）が発生すると、漏れた電流の分だけベクトル和がゼロにならない。

```
Ia + Ib + Ic = I₀（地絡時）
```

この **I₀（零相電流）** を検出するのがZCT。

### 動作原理

1. 三相の電線をZCTの穴に一括貫通させる
2. 正常時：三相の磁束が打ち消し合い、二次側電流ゼロ
3. 地絡時：零相電流による磁束が発生し、二次側に電流が流れる
4. この電流を継電器で検出して地絡を判断

### 構造

| 部位 | 説明 |
|------|------|
| 鉄心 | リング状、三相一括貫通用 |
| 二次巻線 | 継電器に接続 |
| ケース | 絶縁保護 |

---

## 種類

### 用途による分類

| 種類 | 用途 |
|------|------|
| 高圧回路用ZCT | GR、DGR用（高圧側の地絡検出） |
| B種接地線用ZCT | 低圧側の漏電監視 |

### 設置位置による使い分け

**高圧回路用ZCT**

- 設置位置：高圧ケーブルを一括貫通
- 組み合わせ：GR、DGR
- 検出対象：高圧側の地絡

**B種接地線用ZCT**

- 設置位置：変圧器のB種接地線を貫通
- 組み合わせ：絶縁監視装置（Esnavi等）
- 検出対象：低圧側の漏電

## 月次点検のポイント

### チェックリスト

| No. | 確認項目 | 確認内容 |
|-----|---------|---------|
| 1 | 外観 | 損傷、変形、腐食がないか |
| 2 | 貫通状態 | 電線が正しく貫通しているか |
| 3 | 二次配線 | 断線、緩みがないか |
| 4 | 取付状態 | 固定が確実か |

### 貫通状態の確認

**正しい貫通**

- 三相の電線が全てZCTの穴を通っている
- 接地線がZCTを通っていない（高圧回路用の場合）

**誤った貫通の例**

- 一部の相が貫通していない → 誤動作の原因
- 接地線が一緒に貫通している → 検出できない

### B種接地線用ZCTの確認

変圧器のB種接地線がZCTを正しく貫通しているか確認する。

**確認ポイント**

- B種接地線がZCTの穴を通っていること
- 他の配線が一緒に通っていないこと
- 接続が確実なこと

## 継電器との組み合わせ

| 組み合わせ | 保護内容 |
|-----------|---------|
| ZCT + GR | 地絡保護（無方向） |
| ZCT + DGR | 方向性地絡保護 |
| ZCT + 絶縁監視装置 | 低圧側漏電監視 |

### GR（地絡継電器）との接続

ZCTで検出した零相電流をGRに入力し、設定値を超えると遮断器をトリップさせる。

### DGR（方向性地絡継電器）との接続

ZCTの零相電流と、ZPD（またはGPT）の零相電圧を組み合わせて、事故方向を判別する。

---

## CTとの違い

| 項目 | CT | ZCT |
|------|-----|-----|
| 測定対象 | 各相の電流 | 零相電流（三相一括） |
| 貫通方式 | 各相個別 | 三相一括 |
| 用途 | 電流測定、過電流保護 | 地絡検出 |
| 二次側開放 | 厳禁（高電圧発生） | 危険度は低いが推奨しない |

---

## トラブル事例

### 誤動作の原因

| 原因 | 対策 |
|------|------|
| 貫通忘れ | 工事後に必ず確認 |
| 誤貫通（接地線が一緒） | 正しい貫通方法を確認 |
| 外部磁界の影響 | 設置位置の見直し |
| 二次配線の断線 | 配線状態を確認 |

### 不動作の原因

| 原因 | 対策 |
|------|------|
| ZCT故障 | 継電器試験で確認 |
| 継電器故障 | 継電器試験で確認 |
| 配線誤り | 配線を確認 |
