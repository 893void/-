# SC（進相コンデンサ）

## 概要

**SC（Static Capacitor）** は、力率を改善するために設置するコンデンサ。

遅れ力率を進み方向に補正し、電力損失の低減と電気料金の節約に貢献する。

---

## 基礎知識

### 力率とは

**力率（Power Factor）** は、電力がどれだけ有効に使われているかを示す指標。

```
力率 = 有効電力(W) / 皮相電力(VA)
```

- 力率1（100%）：理想的、すべての電力が有効に使われている
- 力率が低い：無効電力が多く、効率が悪い

### なぜ力率が下がるか

モーター、変圧器などの誘導性負荷は「遅れ電流」を取る。

- 遅れ電流 → 遅れ力率 → 効率低下

### 進相コンデンサの役割

コンデンサは「進み電流」を取る。

- 進み電流で遅れ電流を打ち消す
- 結果として力率が改善される

---

## 力率改善のメリット

| メリット | 内容 |
|---------|------|
| 電気料金の低減 | 力率割引の適用（基本料金の割引） |
| 電力損失の低減 | 電流が減少し、I²R損失が減る |
| 設備容量の有効活用 | 同じ設備でより多くの有効電力を使える |
| 電圧降下の改善 | 電流減少により電圧降下が減る |

### 力率割引

電力会社との契約で、力率に応じた割引がある。

- 力率85%を基準
- 力率が高いほど割引
- 力率が低いと割増

## 種類

### 構造による分類

| 種類 | 特徴 |
|------|------|
| 油入コンデンサ | 絶縁油を使用、従来型 |
| 乾式コンデンサ | 油を使用しない、環境対応 |

### 開閉方式による分類

| 種類 | 特徴 |
|------|------|
| 手動開閉 | 手動でON/OFF |
| 自動力率調整（APFR） | 力率に応じて自動でON/OFF |

## SR（直列リアクトル）

### なぜSRが必要か

コンデンサを設置すると、電源側の高調波が増幅される可能性がある。

**高調波障害**

- 機器の異常発熱
- 継電器の誤動作
- コンデンサの破損

### SRの役割

**SR（Series Reactor）** をコンデンサと直列に接続し、高調波の増幅を防ぐ。

- 一般的にはコンデンサ容量の6%のリアクトルを使用
- 第5高調波（250Hz）に対する共振を防止

### SC + SR の組み合わせ

```
電源 ── SR ── SC ── 接地
```

## 月次点検のポイント

### チェックリスト

| No. | 確認項目 | 確認内容 |
|-----|---------|---------|
| 1 | 外観 | 損傷、変形、**膨らみ**がないか |
| 2 | 異音 | 異常な音（うなり等）がしていないか |
| 3 | 異臭 | 焦げ臭、油の臭いがしていないか |
| 4 | 温度 | 異常な発熱がないか |
| 5 | 油漏れ | 油入の場合、漏れがないか |
| 6 | 表示灯 | APFR表示が正常か |

### 膨らみの確認

**コンデンサの膨らみは重大な異常信号**

- 内部で絶縁劣化やガス発生が起きている可能性
- 最悪の場合、破裂の危険

膨らみを発見したら、速やかに使用を停止し、交換を検討する。

## 年次点検のポイント

### 月次点検との違い

| 項目 | 月次点検 | 年次点検 |
|------|---------|---------|
| 停電 | 不要 | 必要 |
| 絶縁抵抗測定 | しない | する |
| 容量測定 | しない | する（推奨） |
| 残留電荷確認 | しない | 必須（安全のため） |

### 年次点検での確認項目

| 項目 | 内容 |
|------|------|
| 絶縁抵抗測定 | コンデンサの絶縁抵抗 |
| 外観精密点検 | 膨らみ、変色、油漏れの確認 |
| 放電抵抗確認 | 内蔵放電抵抗の動作確認 |
| SR確認 | 直列リアクトルの状態確認 |

### 残留電荷に注意

**コンデンサは電荷を蓄える**

停電後もコンデンサに電荷が残っている可能性がある。

**対策**

- 停電後、一定時間待つ（放電抵抗で放電）
- 放電棒で残留電荷を放電してから作業
- 検電して無電圧を確認

## 自動力率調整器（APFR）

### APFRとは

**APFR（Automatic Power Factor Regulator）** は、力率を監視し、コンデンサを自動で投入/開放する装置。

### 動作

1. 力率を常時監視
2. 力率が下がる → コンデンサを投入
3. 力率が上がりすぎる → コンデンサを開放
4. 目標力率を維持

### 点検時の確認

- 表示が正常か
- 自動/手動の切り替えスイッチの状態
- 投入/開放が正常に動作するか（年次点検時）

## トラブル事例

### コンデンサ故障の原因

| 原因 | 対策 |
|------|------|
| 過電圧 | 電圧管理、LA設置 |
| 高調波 | SR設置、フィルタ設置 |
| 過熱 | 換気改善、容量見直し |
| 経年劣化 | 定期的な点検、更新 |

### 膨らみ発見時の対応

1. 直ちに開閉器を開放（停電させる）
2. 十分な放電時間を取る
3. 専門業者に連絡
4. 交換を検討
