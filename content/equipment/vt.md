# VT（計器用変圧器）

## 概要

**VT（Voltage Transformer）** は、高電圧を計測可能な低電圧に変換する変成器。

**PT（Potential Transformer）** とも呼ばれる。

高圧回路の電圧を直接測定するのは危険なため、VTで変換して計器や継電器に入力する。

二次側の定格電圧は **110V** が標準。

---

## 基礎知識

### 動作原理

VTは変圧器と同じ電磁誘導の原理を利用している。

1. 一次側に高電圧を印加
2. 鉄心に磁束が発生
3. 二次側に低電圧が誘導される

**変圧比**

```
V₁ / V₂ = N₁ / N₂
```

例：変圧比 6,600/110V のVT → 一次側6,600Vで二次側110V

### 構造

| 部位 | 説明 |
|------|------|
| 鉄心 | 磁束を通す |
| 一次巻線 | 高圧回路に接続 |
| 二次巻線 | 計器・継電器に接続 |
| 二次端子 | 極性あり |
| 銘板 | 変圧比、定格負担等を記載 |

---

## 種類

### 用途による分類

| 種類 | 用途 |
|------|------|
| 計器用VT | 電圧計、電力計への入力 |
| 保護用VT | 継電器への入力 |
| 計器・保護兼用VT | 両方に使用 |

### 特殊なVT

| 種類 | 特徴 |
|------|------|
| GPT（EVT） | 接地形計器用変圧器。零相電圧検出用 |
| VCT | CTとVTを一体化した計器用変成器 |

---

## 二次側短絡厳禁

### なぜ危険か

VTの二次側を短絡すると、以下の現象が起こる。

1. 二次側に過大な電流が流れる
2. VTが焼損する
3. 一次側ヒューズが溶断（設備によっては停電）

### 対策

- 二次側は絶対に短絡しない
- 二次側の配線作業時は必ず一次側を開放
- 使用しない二次巻線は開放しておく

**CTとの違い**

| 機器 | 禁止事項 |
|------|---------|
| CT | 二次側 **開放** 厳禁 |
| VT | 二次側 **短絡** 厳禁 |

## 月次点検のポイント

### チェックリスト

| No. | 確認項目 | 確認内容 |
|-----|---------|---------|
| 1 | 外観 | 損傷、変形、腐食がないか |
| 2 | 端子 | 緩み、変色、発熱がないか |
| 3 | 二次配線 | 断線、緩みがないか |
| 4 | 接地 | 二次側接地が確実か |
| 5 | 電圧値 | 計器の指示が正常範囲か |
| 6 | ヒューズ | 溶断していないか |

### 電圧値の確認

| 状態 | 考えられる原因 |
|------|---------------|
| 指示がゼロ | VTヒューズ溶断、配線断線 |
| 異常に高い/低い | 電源側の異常、VT故障 |
| 三相不平衡 | 欠相、VT故障 |

## VTヒューズ

### 役割

VTの一次側または二次側にはヒューズが設置されている。

- 一次側ヒューズ：VT内部短絡時の保護
- 二次側ヒューズ：二次回路短絡時の保護

### ヒューズ溶断時の影響

- 電圧計がゼロを指示
- 電力計、電力量計が動作しなくなる
- 電圧に関連する継電器（UVR等）が誤動作の可能性

---

## 関連機器との組み合わせ

| 組み合わせ | 用途 |
|-----------|------|
| VT + 電圧計 | 電圧測定 |
| VT + CT + 電力計 | 電力測定 |
| VT + CT + 電力量計 | 電力量計量 |
| VT + UVR | 不足電圧保護 |
| VT + OVR | 過電圧保護 |

---

## CTとの比較

| 項目 | CT | VT |
|------|-----|-----|
| 変換対象 | 電流 | 電圧 |
| 二次側定格 | 5A | 110V |
| 禁止事項 | 二次開放 | 二次短絡 |
| 接続方式 | 直列（電流が貫通） | 並列（電圧を印加） |
