# VCT（計器用変成器）

## 概要

**VCT（Voltage and Current Transformer）** は、CTとVTを一体化した計器用変成器。

電力会社との **取引用計量** に使用される。

---

## 基礎知識

### 役割

1. **高圧の電流を変換**：CT機能
2. **高圧の電圧を変換**：VT機能
3. **取引用計量**：電力量計への入力

### なぜ一体型か

取引用計量では、CTとVTの精度が重要。

- 一体型にすることで精度を保証
- 封印により不正防止
- 電力会社の検定を受けた機器を使用

### 構造

| 部位 | 説明 |
|------|------|
| CT部 | 電流を変換（定格二次電流5A） |
| VT部 | 電圧を変換（定格二次電圧110V） |
| 二次端子箱 | CT・VTの二次端子を収納 |
| 封印 | 不正防止のための封印 |
| 銘板 | 変流比、変圧比、精度等を記載 |

---

## 種類

### 計量方式による分類

| 方式 | 説明 |
|------|------|
| 高圧計量 | 高圧側で計量（VCT使用） |
| 低圧計量 | 低圧側で計量（変圧器二次側） |

### 高圧計量と低圧計量

| 項目 | 高圧計量 | 低圧計量 |
|------|---------|---------|
| 計量位置 | 変圧器一次側 | 変圧器二次側 |
| 計量器 | VCT + 電力量計 | 電力量計のみ |
| 損失の扱い | 需要家負担 | 電力会社負担 |
| 契約容量 | 大容量向け | 小容量向け |

## 月次点検のポイント

### チェックリスト

| No. | 確認項目 | 確認内容 |
|-----|---------|---------|
| 1 | 外観 | 損傷、変形、腐食がないか |
| 2 | 封印 | 封印が破損していないか |
| 3 | 端子箱 | 蓋が閉まっているか |
| 4 | 配線 | 断線、緩みがないか |
| 5 | 電力量計 | 円盤が回転しているか |

### 封印の確認

VCTには電力会社の封印がある。

- 封印が破損していると問題になる
- 発見した場合は電力会社に連絡

**絶対に封印を破らない**

---

## 年次点検のポイント

### 月次点検との違い

| 項目 | 月次点検 | 年次点検 |
|------|---------|---------|
| 停電 | 不要 | 必要 |
| 絶縁抵抗測定 | しない | する（二次側のみ） |
| 清掃 | しない | する |

### 年次点検での確認項目

| 項目 | 内容 |
|------|------|
| 絶縁抵抗測定 | 二次側回路の絶縁抵抗 |
| 外観精密点検 | 碍子、端子の確認 |
| 清掃 | 碍子、外箱の清掃 |
| 配線確認 | 二次配線の状態確認 |

### 注意事項

- VCT本体の試験は電力会社が実施（検定）
- 需要家側で勝手に試験や分解をしない
- 封印を破らない

---

## 電力量計

### 電力量計との接続

VCTの二次側は電力量計に接続される。

```
VCT(CT部) ──→ 電力量計（電流入力）
VCT(VT部) ──→ 電力量計（電圧入力）
```

### 電力量計の確認

| 確認項目 | 内容 |
|---------|------|
| 円盤の回転 | 負荷があれば回転しているはず |
| 表示値 | 前回との差が妥当か |
| 外観 | 損傷、封印の状態 |

---

## 取引計量の仕組み

### 計量の流れ

1. VCTで高圧の電流・電圧を変換
2. 電力量計で電力量を積算
3. 電力会社が検針
4. 電気料金を請求

### 計量精度

取引用計量は高い精度が求められる。

| 項目 | 要求精度 |
|------|---------|
| VCT（CT部） | 0.5級または1.0級 |
| VCT（VT部） | 0.5級または1.0級 |
| 電力量計 | 1.0級または2.0級 |

### 検定

- VCTと電力量計は定期的に検定を受ける
- 検定は電力会社または検定機関が実施
- 有効期限がある（通常7年または14年）

---

## トラブル事例

### VCT関連のトラブル

| トラブル | 原因 |
|---------|------|
| 電力量計が止まっている | VT部ヒューズ溶断、配線断線 |
| 計量値がおかしい | VCT故障、配線誤り |
| 封印が破損 | 外部からの衝撃、不正 |

### 発見時の対応

- 電力会社に連絡
- 勝手に修理や調整をしない
- 記録を残す
